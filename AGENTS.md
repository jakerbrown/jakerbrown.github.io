# jakerbrown.github.io guidance

This repo hosts the public site and a small amount of shared Codex workflow
infrastructure.

## Codex diary breadcrumbs

For any non-trivial Codex work session in this repo, leave a short breadcrumb
entry under `logs/codex_activity/` before wrap-up, using a filename like
`YYYY-MM-DD_short-task.md`.

Each breadcrumb should briefly capture:

- what you worked on
- what changed
- why it mattered
- what was verified or left unresolved

Write in plain English. Keep it concise, concrete, and not overly technical.
These breadcrumbs are collected by the nightly diary automation in this repo,
so do not leave the only useful summary in chat.
